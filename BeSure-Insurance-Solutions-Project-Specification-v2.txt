1. Project Overview
Myloe Insurance Agency is a web-based insurance agency management system built for the Kenyan insurance market, regulated by the Insurance Regulatory Authority (IRA). Motor vehicle insurance is the primary class, with architecture designed to accommodate Fire & Perils, Domestic, Medical, Life, and Travel insurance as the agency grows.

Module 3: Premium Calculator
The calculator is both embedded in the policy creation flow (Steps 4-7) and available as a standalone tool at /calculator for quick quotes without saving a policy.

3.1 Inputs (all manual)
• Insurance Type (Motor Private / Commercial / PSV)
• Cover Type (Comprehensive / TPO / TPFT)
• Insurer (dropdown - auto-fills the rate field)
• Vehicle Value / Sum Insured (KES) - typed by staff
• Basic Premium Rate (%) - auto-filled from insurer default; editable
• Selected Additional Benefits with manual KES amounts each

3.2 Outputs (all calculated live)
• Basic Premium = Vehicle Value × Rate ÷ 100
• Minimum Premium check: if calculated basic premium is below insurer minimum, apply the minimum and flag it
• IRA Levy = 0.45% × Basic Premium
• Training Levy = 0.2% × Basic Premium
• Stamp Duty = KES 40 (fixed)
• PHCF = KES 100 (fixed)
• Total Benefits = sum of all manually entered benefit amounts
• Grand Total Premium = Basic Premium + Total Benefits + IRA Levy + Training Levy + Stamp Duty + PHCF
• Agency Commission = Grand Total × Commission Rate% (from insurer settings) - shown separately for internal reference

4.2 Installment Schedule Rules
• First installment: due on policy start date (staff can override)
• Subsequent installments: spaced 30 days apart by default (configurable in Settings)
• Staff can manually adjust any individual due date after the schedule is created
• Each installment records: amount due, amount paid, date paid, payment method, reference number, outstanding balance
• Outstanding balance calculated in real time across all installments

6. Recommended Build Phases
Phase 1 - Foundation (Weeks 1-2)
• Next.js project setup, Neon DB, Drizzle ORM, Tailwind, shadcn/ui
• NextAuth single admin login
• Deploy to Vercel, configure environment variables
• Database schema creation and Drizzle migrations
• Customer module: list, create, view, edit, document upload

Phase 2 - Policy & Calculator (Weeks 3-4)
• Settings: Insurer management (name, rates, minimums, commission)
• Settings: Benefits Manager (add/rename/deactivate benefit options)
• Settings: Garage Register
• 7-step policy creation stepper with all insurance types
• Premium calculator (standalone + embedded in policy flow)
• Logbook and valuation report file uploads on policy

Phase 3 - Payments & Notifications (Weeks 5-6)
• Payment module: full, 2-part, 3-part installment, IPF
• Auto-split installment amounts and due date scheduling
• Payment receipt PDF generation
• Resend email integration
• Vercel Cron Job: daily 10 / 3 / 2 day payment reminders + expiry alerts
• Notification log

Phase 4 - Cover Tracking & Claims (Weeks 7-9)
• Cover tracking pipeline and document tracker per policy
• Turnaround time calculations and SLA breach flags
• Follow-up notes log (polymorphic: policy / claim / customer)
• Claims module: create, stage management, document tracker
• Garage updates and parts follow-up log
• Claims summary PDF
